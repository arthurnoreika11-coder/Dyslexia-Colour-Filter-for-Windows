# Dyslexia Colour Filter for Windows

## Why build this when you can by something?
### Because say what you will about Apple, it has it built in: 
To set up color filters on a Mac (macOS Ventura or later), go to System Settings > Accessibility > Display and toggle on Color Filters. You can choose from filters like Greyscale, Red/Green, or a customized Color Tint. Use the [Option-Command-F5] shortcut to toggle them on or off quickly.

But also because I am an aspiring Legal Technologist / Legal Engingeer wanting to relearn coding. A personal Windows accessibility and legal technology learning project for a subtle coloured screen overlay to support dyslexia, visual stress, and more comfortable on-screen reading.

I am building this from lived experience as a dyslexic person who can struggle with extended on-screen reading. 

The repository currently contains a PowerShell prototype. The intended long-term version is a modern C# WPF desktop app that demonstrates accessibility thinking, Windows desktop development, and responsible software design in a law firm context.

This is not intended to bypass workplace IT controls. If used on a work laptop, it should be shared with IT/security or workplace adjustments teams for review first.
